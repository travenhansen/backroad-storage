BACKROAD STORAGE STATIC WEBSITE

Files:
- index.html
- storage-units.html
- parking-storage.html
- contact.html
- styles.css
- script.js
- assets/...

WHAT TO EDIT FIRST
1. Open script.js
2. Change availabilityStatus to 'available' or 'full'
3. Update availableUnitsCount and availableParkingCount when needed
4. Paste your TenantCloud booking link into tenantCloudUrl later

WAITLIST FORM
- Open contact.html
- Replace https://formspree.io/f/YOUR_FORM_ID with your actual form endpoint
- Formspree is the easiest option for a static site

SEO NOTES
- Update the canonical URLs after you connect your real domain
- Keep your Google Business Profile updated
- Add your Google review link to the site later if you want a reviews section

HOSTING OPTIONS
- Netlify
- Vercel
- GitHub Pages
- Any basic web host
